# PRIMEURO Landing Page

Professional landing page for PRIMEURO in-game services agency.

## How to Add Your Images

### Hero Section Image
1. Place your hero image in the same folder as `index.html`
2. Name it `hero-image.jpg` (or update line 93 in `index.html` with your image filename)
3. Recommended size: 1200x600px or similar 2:1 ratio

### Service/Product Images
1. Place your product images in the same folder as `index.html`
2. Update the following lines in `index.html`:
   - Line 216: `service-currency.jpg` - Premium Currency service image
   - Line 244: `service-items.jpg` - Exclusive Items service image
   - Line 273: `service-boosting.jpg` - Boosting Services image
3. Recommended size: 600x400px or similar landscape ratio

### Uploading Images
Simply place your image files in the `/home/user/priv/` directory with the names above, or edit the `src` attributes in the HTML file to match your image filenames.

## Mobile Optimization

This site is **mobile-first** and optimized for mobile users:
- Large, touch-friendly buttons (48px+ height)
- Readable text sizes (16px minimum)
- Full-width buttons on mobile devices
- Optimized image loading and display
- Responsive service cards that stack on mobile
- Easy navigation with mobile menu
